# Penetration Testing Foundation

Hands-on foundational penetration testing training covering web application assessment, infrastructure testing, attack methodology, reporting workflows, and industry-standard pentesting frameworks.

## Guided Web Application Penetration Testing
Practical attack chain execution across a simulated web application environment.

Coverage:
- reconnaissance
- enumeration
- IDOR exploitation
- weak password reset assessment
- admin panel discovery
- remote code execution
- chained attack methodology
- attack path progression

Practical concepts reinforced:
- attack surface mapping
- access control testing
- authentication weakness analysis
- privilege escalation opportunities
- web exploitation workflow
- attack chaining mindset
