# Enumeration Methodology Notes

Enumeration is one of the most critical phases in offensive security and penetration testing.

This section documents practical enumeration workflows, techniques, and lessons learned.

Areas of focus:

- Host discovery
- Port scanning
- Service enumeration
- Banner grabbing
- Web directory discovery
- Subdomain enumeration
- DNS enumeration
- SMB enumeration
- HTTP inspection
- Reconnaissance methodology

Tools commonly used:

- Nmap
- Gobuster
- ffuf
- Netcat
- curl
- enum4linux
