- Information gathering
- Active / passive reconnaissance
- Network scanning
- Enumeration
- Host and network assessment
- Web application security fundamentals
- Exploitation basics
- Authentication / Authorization concepts

- Pentesting methodology

- John the Ripper
- Metasploit Framework
- Netcat
- Linux command line
